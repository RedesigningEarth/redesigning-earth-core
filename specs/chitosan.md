# Chitosan — Material Specification (Extended)

## Chemical identity
- Polymer: Poly-(1→4)-2-amino-2-deoxy-β-D-glucan
- CAS: 9012-76-4

## Physical & thermal properties (summary)

| Property | Typical range | Notes / relevance |
|---|---|---|
| Apparent density | 0.3–0.5 g/cm³ | Powder |
| True density | 1.3–1.4 g/cm³ | Literature |
| Solubility | Insoluble in water | Soluble in dilute acids |
| pH | 6.5–8.5 | Aqueous slurry |
| Melting behavior | None | No true melt |
| Decomposition onset | >185 °C | Upper processing limit |
| Glass transition | Not defined | Solvent/moisture dependent |

## Mechanical relevance (literature ranges)

| Property | Order of magnitude | Dependency |
|---|---|---|
| Elastic modulus | 1–4 GPa | DDA, moisture |
| Tensile strength | 30–100 MPa | MW, bonding |
| Elongation at break | <10% | Dry state |

## Experimental result (RE)
- Heat + shear plasticization at ~190 °C: **disproven**

## Safety summary (SDS)
- Non-hazardous
- Toxic fumes when burning
- PPE: respirator, gloves, goggles
