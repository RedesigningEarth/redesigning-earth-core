# Chitin — Material Specification (Extended)

## Chemical identity
- Polymer: β-(1→4)-linked N-acetyl-D-glucosamine
- Empirical repeat unit: C6H11NO4
- CAS: 1398-61-4

## Physical & thermal properties (summary)

| Property | Typical range | Notes / relevance |
|---|---|---|
| Apparent density (powder) | 0.3–0.6 g/cm³ | Packing dependent |
| True density (crystalline) | ~1.4 g/cm³ | Literature |
| Moisture (as supplied) | ≤20% | Supplier; RE target <5% |
| Solubility | Insoluble in water | Soluble in acids |
| pH (slurry) | 7.0–8.0 | Near-neutral |
| Melting behavior | None | Does not melt |
| Decomposition onset | >185 °C | Thermal limit |
| Thermal conductivity | ~0.2 W/m·K | Polysaccharide typical |

## Mechanical relevance (literature ranges)

| Property | Order of magnitude | Context |
|---|---|---|
| Elastic modulus | 2–5 GPa | Crystalline chitin |
| Tensile strength | 50–150 MPa | Natural composites |
| Fracture behavior | Brittle | Dry state |

## Safety summary (SDS)
- Non-hazardous transport
- Emits toxic fumes under fire
- PPE: respirator, gloves, goggles

## Storage & handling
- Cool, dry, ventilated
- Avoid oxidizers and dust formation
